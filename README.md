## Overview

A currency exchange rate checker app built as part of a Frontend Mentor challenge. The app will allow users to look up exchange rates between currencies.

## Getting Started

### Prerequisites

- Node.js ≥ 18
- npm or yarn

### Installation

```bash
# Clone the repo
git clone https://github.com/your-username/fx-checker.git
cd fx-checker

# Install dependencies
npm install

# Start the dev server
npm run dev
```
